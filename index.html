Vocabulaire

Front-End

  • ⯅ Créé l'interface et la partie interactive du site
  • ⯅ Utilise le HTML, le CSS et le JavaScript
  • ⯅ Harmonise le design et l'interactivité
  • ⯅ Assure la compatibilité avec mobiles et tablettes

Back-End

  • ⯅ Créé la partie serveur et traitement
  • ⯅ Langages PHP, Ruby, Node.js, Python...
  • ⯅ mongoDB, PostgreSQL, MySQL..
  • ⯅ Gère aussi la sécurité, la performance, etc.

Full Stack

  • ⯅ Maîtrise le front-end et le back-end
  • ⯅ Gère de la conception à la mise en production
  • ⯅ Conçoit aussi bien l'interface que la partie serveur

Langages

HTML

  • ⯅ Créé en 1993
  • ⯅ Permet de créer la structure des pages web
  • ⯅ Utilise des balises pour structurer le contenu
  • ⯅ Version actuelle: HTML 5

CSS

  • ⯅ Créé en 1995
  • ⯅ Complémentaire au HTML
  • ⯅ Permet de mettre en forme les pages web
  • ⯅ Version actuelle: CSS 3

JavaScript

  • ⯅ Créé en 1995
  • ⯅ Permet de rendre les pages web dynamiques
  • ⯅ Peut modifier dynamiquement le HTML et le CSS
  • ⯅ Large écosystème de frameworks
  • ⯅ Version actuelle: ES6

Outils

IDE

  • ⯅ Outil facilitant la rédaction de code
  • ⯅ Visual Studio Code, Atom, Sublime Text...
  • ⯅ Intègre un éditeur, débogueur et divers outils
  • ⯅ Recommandation: VS Code

Navigateur

  • ⯅ Interprète le code HTML, CSS et exécute le JS
  • ⯅ Firefox ou Chrome recommandés
  • ⯅ Contient généralement des outils développeur
  • ⯅ Peut lire le HTML/CSS/JS localement

Git

  • ⯅ Outil de contrôle de versions
  • ⯅ Facilite la collaboration sur le code
  • ⯅ Compte GitHub nécessaire
  • ⯅ Nécessite d'être installé
